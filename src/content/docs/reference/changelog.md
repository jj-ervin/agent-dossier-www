## PASS Changelog Format

All changes recorded using PASS schema:

```yaml

  principal: "maintainer-name"
  action: "added"            # added, modified, removed, deprecated
  subject: "field_or_concept"
  scope: "section_name"      # identity_block, authorities, constraints, etc.
  reason: "Rationale for change"
```
