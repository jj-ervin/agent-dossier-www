# Agent Dossier Specification Website

Official website for the Agent Dossier Specification: A deterministic, auditable governance standard for agents in multi-agent development ecosystems.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

```bash
cd c:\dev\agent-dossier-www
npm install
```

### Development Server

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

### Preview Build

```bash
npm run preview
```

## 📚 Documentation

The site is built with **Astro** and **Starlight** (a documentation framework).

### Adding Content

Add Markdown files to `src/content/docs/`:

```markdown
---
title: Page Title
description: Page description for metadata
---

# Your Content Here

This is a markdown page.
```

Update `astro.config.mjs` sidebar if needed.

### Structure

The site follows the scope document structure:

- **Getting Started**: Quick intro and quick start guide
- **Documentation**: Core concepts, identity, authorities, constraints, etc.
- **Reference**: Schemas, grammars, changelog
- **Resources**: Examples, adopters, contributing, FAQ

## 🔗 Configuration

Key files:

- `astro.config.mjs` — Site configuration, navigation, metadata
- `package.json` — Dependencies and scripts
- `src/content/config.ts` — Content schema definitions
- `WEBSITE_SCOPE.md` — Project scope document

## 📦 Deployment

The site is designed for GitHub Pages deployment:

1. Push to GitHub
2. Configure GitHub Actions to build and deploy
3. Set CNAME to `agent-dossier.org`
